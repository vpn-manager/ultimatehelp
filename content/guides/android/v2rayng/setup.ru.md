## Добавление ссылки на подписку
1. Откройте бота и с помощью кнопки управления сервисом получите ссылку на подписку. Затем скопируйте ссылку.
<img src="/assets/android/v2rayng/1.png"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/2.png"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>
2. Откройте v2rayNG, нажмите ➕, затем выберите второй пункт Import config from Clipboard.

<img src="/assets/android/v2rayng/3.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/4.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>

3. Нажмите import sub, чтобы появился список серверов.
<img src="/assets/android/v2rayng/5.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/6.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>

4. Выберите сервер с учетом вашего подключения, затем нажмите кнопку в правом нижнем углу, чтобы подключиться.
<img src="/assets/android/v2rayng/7.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>
5. Если вы используете приложение впервые, может появиться такое сообщение. Нажмите OK.
<img src="/assets/android/v2rayng/8.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>
6. Если список серверов пуст, нажмите на три точки в правом верхнем углу и выберите Update subscription, чтобы добавить серверы.
<img src="/assets/android/v2rayng/10.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/14.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>



:::warning

**⚠️ Внимание:**
1. Не держите другой VPN подключенным при добавлении ссылки для обновления.

2. Убедитесь, что дата и время на телефоне установлены автоматически.
:::
