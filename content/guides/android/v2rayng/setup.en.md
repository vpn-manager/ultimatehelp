## Adding a Subscription Link
1. Open the bot and use the service management button to get your subscription link. Then copy the link.
<img src="/assets/android/v2rayng/1.png"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/2.png"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>
2. Open v2rayNG, tap ➕, then select the second option, Import config from Clipboard.

<img src="/assets/android/v2rayng/3.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/4.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>

3. Tap import sub so the server list is shown.
<img src="/assets/android/v2rayng/5.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/6.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>

4. Choose a server based on your connection, then tap the button in the bottom-right corner to connect.
<img src="/assets/android/v2rayng/7.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>
5. If this is your first time using the app, you may see this message. Tap OK.
<img src="/assets/android/v2rayng/8.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>
6. If your server list is empty, tap the three dots in the top-right corner and select Update subscription to add the servers.
<img src="/assets/android/v2rayng/10.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<img src="/assets/android/v2rayng/14.jpg"
     style="display:block;float:none;margin-left:auto;margin-right:auto;width:52%">
<br>



:::warning

**⚠️ Note:**
1. Do not keep another VPN connected while adding the update link.

2. Make sure your phone's time and date are set to automatic.
:::
